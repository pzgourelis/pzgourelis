## Featured Project

### Single-Use Plastics: Questionnaire-Based Excel and SQL Analysis

An academic data-analysis project examining single-use plastic consumption, environmental awareness, reusable alternatives, behavioral intentions, and perceived barriers among residents of Grevena, Greece.

The project follows a complete data workflow:

**Questionnaire design → Excel coding and validation → Relational database implementation → SQL analysis → Interpretation and multilingual web presentation**

The repository includes:

- a structured academic questionnaire
- an anonymized Excel dataset
- a variable dictionary and coding guide
- a normalized SQL Server database
- SQL scripts for data insertion, validation, and analysis
- a complete academic report
- a multilingual website available in English, Greek, and German
